# 03. UI & 커스터마이징 설계

## 위젯 기본 디자인

```
┌──────────────────────────────┐
│ ● Claude          14:32 갱신 │
│ 세션   ▓▓▓▓▓▓▓░░░  68%       │
│        2시간 14분 후 초기화   │
│ 주간   ▓▓▓▓░░░░░░  41%       │
│        금요일 오전 9:00 초기화│
└──────────────────────────────┘
```

- 둥근 모서리(radius 16px), 반투명 배경 + blur 느낌
- 프로그레스 바에 그라데이션 fill
- 사용률 구간별 색: 정상(테마색) / 80%+ 주황 / 95%+ 빨강
- 컴팩트 모드: 바 2개 + %만 (한 줄 높이)

## 레이아웃 3종 (설정에서 클릭 전환)

1. **가로형** (기본): 위 그림. 280×130px 내외
2. **세로형**: 좁고 긴 형태, 사이드에 붙이기 좋음
3. **컴팩트**: 바+% 만, 라벨 최소화

## 설정 패널 구성

위젯 우클릭 또는 트레이 메뉴 → 설정 창 (탭 3개):

- **테마**: 프리셋 그리드(클릭 즉시 적용) / 커스텀 색상 / 다크·라이트 배경 / 투명도 슬라이더(30–100%)
- **레이아웃**: 가로·세로·컴팩트 / 크기 S·M·L / 표시 항목 체크박스(세션·주간·모델별·계정명·갱신시각)
- **동작**: 폴링 주기 / 경고 임계값 / Windows 알림 on·off / 부팅 시 시작 / 항상 위 on·off

모든 변경은 저장 버튼 없이 즉시 위젯에 반영 (IPC로 push).

## 설정 스키마 (settings.json)

```json
{
  "theme": {
    "preset": "claude",
    "customGradient": { "from": "#D97757", "to": "#E8A87C", "angle": 135 },
    "background": "dark",
    "opacity": 0.92
  },
  "layout": { "mode": "horizontal", "size": "M",
              "fields": ["session", "weekly", "resetTime"] },
  "behavior": { "pollIntervalSec": 90, "warnAt": 80, "dangerAt": 95,
                "notifications": true, "autoStart": false, "alwaysOnTop": true },
  "position": { "x": null, "y": null }
}
```

## 구현 방식

- 테마는 CSS 변수로 일원화: `--grad-from`, `--grad-to`, `--grad-angle`, `--bg`, `--opacity`
- 설정 변경 → main이 settings 저장 → 위젯 렌더러에 IPC → CSS 변수만 교체 (리로드 없음)
- 프리셋 추가가 쉽도록 프리셋 목록을 JSON 상수로 분리
