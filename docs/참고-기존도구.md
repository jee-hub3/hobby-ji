# 참고: 지금 바로 쓸 수 있는 기존 도구

자체 개발 완성 전까지 바로 사용 가능. 요구사항(세션·주간 사용량 + 초기화 시간, 항상 표시)을 이미 충족하는 오픈소스들.

## ⭐ 추천: Usage Monitor for Claude

https://github.com/jens-duttke/usage-monitor-for-claude

- Windows 트레이 앱, **단일 exe(~12.5MB), 설치·설정 불필요**
- Claude Code 로그인 정보 자동 사용 (우리 기획과 동일한 방식)
- 트레이 아이콘에 세션+주간 바 실시간 표시, 클릭하면 상세 팝업(모든 한도 + 리셋 카운트다운)
- 한국어 지원, 임계값 알림, 토큰 만료 시 자동 갱신
- MIT, 코드 감사 용이 (api.anthropic.com 단일 통신)

**실행법**:
1. https://github.com/jens-duttke/usage-monitor-for-claude/releases/latest 에서 `UsageMonitorForClaude.exe` 다운로드
2. 아무 폴더에 두고 더블클릭 (SmartScreen 경고 시 "추가 정보 → 실행")
3. 트레이 아이콘 확인. 안 보이면: 작업표시줄 우클릭 → 설정 → 기타 시스템 트레이 아이콘 → 켜기

**한계 (= 우리가 만들 이유)**: 트레이 중심이라 화면에 항상 떠 있는 위젯이 아니고, 색상은 JSON 파일로만 변경 가능 — 그라데이션·배치 클릭 커스터마이징 없음.
