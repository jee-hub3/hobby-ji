# 01. 요구사항 명세

## 목표

Claude 사용량을 항상 화면 위에 표시하는 가볍고 예쁜 데스크톱 위젯.
"내가 지금 얼마나 썼고, 언제 초기화되는지"를 한눈에.

## 사용자 스토리

- 작업 중 화면 구석의 위젯만 봐도 세션/주간 사용량과 초기화까지 남은 시간을 알 수 있다
- 위젯 색상(그라데이션), 배치(가로/세로/컴팩트), 투명도를 설정 화면에서 클릭 몇 번으로 바꿀 수 있다
- 한도에 가까워지면 색상 변화·알림으로 미리 알 수 있다
- 다른 사람은 GitHub에서 exe 하나만 받아 실행하면 된다 (Claude Code 로그인만 되어 있으면 설정 불필요)

## 기능 명세

### MVP (Phase 1–2)

| ID | 기능 | 설명 |
|---|---|---|
| F-01 | 세션 사용량 표시 | 5시간 롤링 세션 사용률(%) 프로그레스 바 |
| F-02 | 주간 사용량 표시 | 7일 한도 사용률(%) 프로그레스 바 (모델별 한도가 있으면 함께) |
| F-03 | 초기화 카운트다운 | 각 한도의 리셋 시각을 "2시간 14분 후 (오후 3:00)" 형식으로 표시 |
| F-04 | 항상 위 표시 | 프레임리스, always-on-top, 드래그로 위치 이동, 위치 기억 |
| F-05 | 자동 갱신 | 1–3분 주기 폴링, 갱신 실패 시 마지막 데이터 + 오래됨 표시 |
| F-06 | 계정 자동 연동 | Claude Code 토큰 자동 감지. 없으면 안내 메시지 표시 |

### v1.0 (Phase 3)

| ID | 기능 | 설명 |
|---|---|---|
| F-07 | 테마 커스터마이징 | 그라데이션 프리셋 6종+, 커스텀 색상 피커, 다크/라이트 |
| F-08 | 레이아웃 커스터마이징 | 가로형 / 세로형 / 컴팩트(바만) 3종, 크기 조절, 투명도 슬라이더 |
| F-09 | 설정 패널 | 위젯 우클릭 → 설정 창. 변경 즉시 미리보기 반영 |
| F-10 | 시스템 트레이 | 트레이 아이콘(위젯 숨기기/보이기, 설정, 종료), 부팅 시 자동 시작 옵션 |
| F-11 | 임계값 경고 | 80%/95% 도달 시 바 색상 변경 + Windows 알림 (임계값 조정 가능) |

### v1.1 (배포 후 개선 후보)

- 시간 대비 사용 속도 마커 (시간 경과보다 빨리 쓰고 있으면 경고색)
- 사용량 히스토리 그래프
- macOS/Linux 지원
- 다국어 (한국어/영어)

## 비기능 요구사항

- 메모리 상주 부담 최소화 (백그라운드 폴링만, 유휴 시 폴링 정지)
- 토큰 보안: 외부 전송 금지, `api.anthropic.com` 단일 통신 대상
- 네트워크 끊김·API 오류 시 크래시 없이 표시 유지
- 설정은 JSON 파일로 저장 (Electron userData 경로)
